# Armazenador Local

O Armazenador Local é um aplicativo desenvolvido em React (para estudo sobre LocalStorage) que permite armazenar informações localmente no navegador do usuário. Ele utiliza `localStorage` previamente mencionado para persistir os dados mesmo após o fechamento do navegador.

## Funcionalidades

- Armazenamento Local: O aplicativo permite armazenar informações localmente no navegador do usuário como: Nome, Sobrenome e Data de Nascimento.
- Visualização de Dados: Os dados armazenados são exibidos em uma grade na interface do usuário.
- Atualização automática: O armazenador local atualiza automaticamente o armazenamento local sempre que os dados são modificados.

## Instalação

1. Clone o repositório do projeto:
    ```bash
        git clone https://github.com/luvsscorpius/ArmazenadorLocal
    ```
2. Instale as dependências do projeto
    ```bash
        cd ArmazenadorLocal
        npm install
    ```

## Uso

1. Inicie o servidor de desenvolvimento:
    ```bash
        npm start
    ```
2. O aplicativo será aberto automaticamente em seu navegador padrão.

## Tecnologias Utilizadas

- React 
- CSS (utilizando Styled Components)
